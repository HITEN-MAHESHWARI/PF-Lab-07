## LAB OBJECTIVES
* Understand the concept of one-dimensional arrays in C.
* Learn how to declare and initialize arrays.
* Understand array indexing and element access.
* Study memory representation of arrays in contiguous locations.
* Learn how to modify array elements using loops.
* Understand the concept of character arrays (strings) in C.
* Practice accessing and modifying characters in a string.
* Learn how to use scanf scanset %[] for specific input.
* Understand negating a scanset ^ to exclude characters.
* Use loops with arrays to process multiple data elements efficiently.
